# IMPROVEMENTS_SPECS.md - ShopHub Improvements Specifications

## Main Goal
- Implementar mejoras en los archivos existentes del proyecto ShopHub.
- Permitir al usuario elegir (UI amigable) que nivel de prioridad para mejoras desea implementar.

## Options

**1. Mejoras de prioridad alta**: Problemas críticos que afectan la funcionalidad o la experiencia del usuario;  Ejecutar las especificaciones de `high_priority.md`.
**2. Mejoras de prioridad media**: Mejoras importantes que optimizan la calidad y el mantenimiento del código;  Ejecutar las especificaciones de `mid_priority.md`.
**3. Mejoras de prioridad baja**: Mejoras y optimizaciones deseables;  Ejecutar las especificaciones de `low_priority.md`.
**4. Volver**: Volver al menú principal.

## Specifications

### Code Style
- **Politica de comentarios**: Usar JsDocs.
