# Mercado Libre Frontend Challenge

Welcome to the Mercado Libre coding challenge! In this challenge, you will be working with a legacy e-commerce application that needs improvement and modernization.

## Challenge Overview
**Platform:** HackerRank

## Main Objective
Transform the provided legacy e-commerce application into a performant, high-quality application by completing all the tasks below.

---

## Solution

- File `specification.md`
- Directory
```
specs/
```

## Tasks
Please complete the following tasks in the specified order:

### Task #1: Create Application Specification
Read through the provided codebase thoroughly and create a file named `specification.md` that precisely explains to a generative AI model or code assistance tool how to build this legacy application from scratch.

**Requirements:**
- Document only the features that currently exist (do not add new features)
- Include technical architecture, component structure, and functionality
- Ensure the specification is clear enough for recreation from scratch

### Task #2: Prioritize and Implement Improvements
Create a folder named `improvements` containing three markdown files that categorize necessary fixes and improvements:
- `high_priority.md` - Critical issues that impact functionality or user experience
- `mid_priority.md` - Important improvements that enhance code quality and maintainability
- `low_priority.md` - Nice-to-have enhancements and optimizations

**Critical Requirement:** You must implement ALL improvements you identify in your markdown files. Each file should contain:
- Detailed specifications for AI tools to implement the fixes
- The actual implemented code changes
- Clear documentation of what was fixed and why

### Task #3: Design a New Feature
Create a file named `new_feature.md` containing a specification for a new feature that would add maximum value with minimal development effort.

**Requirements:**
- Justify why this feature adds significant value
- Explain why the implementation effort is minimal
- Provide detailed implementation specifications for AI tools
- Include the actual implemented feature code

## Submission Requirements
- Submit all files through the HackerRank platform
- Ensure all code runs without errors
- Include clear documentation for each deliverable

## Getting Started

### Prerequisites
- Node.js 18.x or 20.x LTS
- npm 8.x or higher

### Running the Application

**Installation:**
```bash
npm install
```

**Development Server:**
```bash
npm run dev
```

**Tests:**
```bash
npm test
```
